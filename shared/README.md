# Shared resources

Cross-course materials that aren't tied to a single module. Examples:

- `cheatsheets/`: Big-O, regex, Git, shell, common algorithms
- `study-habits/`: Spaced repetition setup, revision schedule templates
- `glossary.md`: Course-spanning CS vocabulary

Keep things here **only** if they genuinely apply across courses. If it's tied to one module, it belongs under `courses/<CODE>/knowledge-base/` instead.
