## Topics covered
1. Identification and classification of cyber threats
2. Cybersecurity frameworks, standards, legal, ethical and professional considerations
3. Network and OS security
4. Application security principles, testing and debugging
5. Cryptography in practice (encryption, PKI, digital signatures, hash functions)
6. Blockchain for cybersecurity
7. Incident response and forensics
8. Emerging trends (AI, ML, quantum computing, post-quantum cryptography, IoT)
9. EDI principles, sustainable digital ecosystems, and employability
10. Real-world cybersecurity case studies
